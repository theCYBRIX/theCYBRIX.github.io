Out Of The Gutter

Unfinished Hackathon

Slimes in a sewer system compete against each other in a variety of minigames such as racing to the exit and hot potato. Out Of The Gutter is a peer-to-peer multiplayer game made using Godot. The game was conceived and brought to it's current state during a hackathon as a collaboration between Koloss0, nigerianpickle, Kingjoker4life, JordonH03 and myself.

In it's current state, the game has functional networking, LAN session discovery and a working but incomplete sewer race mode.

Clock Racer

Unfinished Hackathon

A multiplayer game where snails work together to pass through obstacle courses within a time limit. Each snail is given a different special ability and the players must work together, making use of their unique abilities, in order to proceed through each level. The game is a peer-to-peer multiplayer game made using Godot.

Clock racer was conceived and brought to it's current state during a hackathon as a collaboration between KarinP03, tsuki0272, mcinn-ng, Candleire and myself.

In it's current state, the game has functional networking and partially completed first level.

SimpleNeuralNetwork

Main Project Ongoing

A neural network framework written in Java featuring a basic evolutionary training algorithm, JSON serialization and deserialization and a custom API for inter-process communication.

The framework is written from the ground up with minimal dependencies and currently does not support GPU compute.

NeuralRaceCars

Main Project (Linked) Ongoing

A 2D top-down racing game written in Godot with the goal of training neural networks to race and drift cars around a track.

NeuralRaceCars integrates with the SimpleNeuralNetwork API and has already had success in training neural networks to both race and drift, but still requires a lot of polish.
